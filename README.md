# AutoExpreso Toll & Expense Tracker

## Description
The AutoExpreso Toll & Expense Tracker is a dedicated, streamlined web application designed specifically for drivers in Puerto Rico to monitor, log, and audit their electronic toll highway expenses. Given the fast-paced nature of modern transit, keeping an accurate record of mobile transponder balances, toll costs, and account recharges can be challenging. This application bridges that gap by offering an intuitive platform where users can manually log every highway transaction, track toll plaza locations, record account replenishment funds, and generate instant digital expense receipts. By centralizing transit costs, commercial fleet drivers and daily commuters alike can analyze monthly travel overheads, prevent unexpected account suspensions, and eliminate the stress of tracking administrative transport fees. Built with responsiveness and user autonomy in mind, this tool offers a structured local ecosystem to organize financial data without relying on cumbersome paper receipts. It serves as an essential companion utility for modern highway navigation across the island's primary tollways.

## Features
* **Real-Time Expense Logging:** Seamlessly add toll costs, select specific highway routes, and classify transponder recharges.
* **Interactive Transaction Ledger:** Display dynamically updated lists of all toll historical logs with categorical filtering options.
* **Digital Receipt Generator:** Render clean, downloadable visual digital receipts for reimbursement or corporate filing.
* **Dynamic Analytics Cards:** Monitor total money spent and current estimated virtual balances instantly at a single glance.
* **Progressive Web App Architecture:** Responsive design crafted beautifully for both desktop dashboards and mobile phone viewports.

## Technologies Used
* **Frontend Library:** React.js (v18+)
* **Styling & Layout:** Vanilla CSS3 with Flexbox and Grid CSS
* **State Management:** React Hooks (`useState`, `useEffect`, `useMemo`)
* **Testing Suite:** Jest & React Testing Library
* **Build System:** Node Package Manager (npm) & Create React App structural backbone
